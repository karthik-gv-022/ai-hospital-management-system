Backend Setup (FastAPI)

1

Clone Repository

# Clone the repository
git clone https://github.com/karthik-gv-022/ai-hospital-management-system.git

# Navigate to project directory
cd ai-hospital-management-system
2

Setup Virtual Environment

# Navigate to backend directory
cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate

# On macOS/Linux:
source venv/bin/activate
3

Install Dependencies

Quick Install (Essential)
# Essential packages only
pip install fastapi uvicorn sqlalchemy pydantic pydantic-settings python-jose passlib bcrypt python-multipart python-dotenv
Full Install (All Features)
# Install all dependencies including ML packages
pip install -r requirements.txt
4

Configure Environment

# Copy environment template
cp .env.example .env

# Edit .env file with your database credentials
nano .env  # or use any text editor
üìù Environment Variables
# Database Configuration
DATABASE_URL=sqlite:///./hospital.db  # SQLite for development
# DATABASE_URL=mysql+pymysql://user:password@localhost:3306/hospital_db  # MySQL for production

# JWT Configuration
SECRET_KEY=your-secret-key-here-change-in-production
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=1440

# Application Configuration
APP_NAME=AI Hospital Management System
DEBUG=True
ENVIRONMENT=development
5

Start Backend Server

Development Mode
# Start with auto-reload
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
Quick Test Server
# Simple test server (for demo)
python test_api.py
‚úÖ Backend URLs
  • ‚Ä¢ API Base URL: http://localhost:8000
  • ‚Ä¢ API Documentation: http://localhost:8000/docs
  • ‚Ä¢ Health Check: http://localhost:8000/health

Frontend Setup (React)

1

Verify Node.js Installation

# Check Node.js version (should be 16+)
node --version

# Check npm version
npm --version

# If not installed, download from: https://nodejs.org/
2

Navigate to Frontend Directory

# From project root, navigate to frontend
cd frontend
3

Install Dependencies

# Install all frontend dependencies
npm install
‚ö†Ô∏è If Installation Fails

Try these alternatives:

# Clear npm cache

# Install with legacy peer deps

4

Configure API Endpoint

The frontend is configured to connect to the backend at http://localhost:8000 by default.

  "proxy": "http://localhost:8000"

Start Frontend Server

Development Mode
# Start React development server
npm start
Alternative: Simple HTTP Server
# Go back to project root
cd ..

# Start simple HTTP server for demo
python -m http.server 3000

Troubleshooting Guide

üîß Common Issues & Solutions

Port Already in Use

Error: Port 8000 is already in use

# Find process using port
lsof -i :8000  # macOS/Linux
netstat -ano | findstr :8000  # Windows

# Kill process
kill -9 [PID]  # macOS/Linux
taskkill /PID [PID] /F  # Windows
Module Not Found

Error: ModuleNotFoundError: No module named 'fastapi'

# Ensure virtual environment is activated
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate     # Windows

# Install dependencies
pip install fastapi uvicorn sqlalchemy pydantic

# Or install all requirements
pip install -r requirements.txt
Database Connection Error

Error: Can't connect to MySQL server

# Use SQLite for development
DATABASE_URL=sqlite:///./hospital.db

# Or check MySQL service
sudo systemctl status mysql  # Linux
brew services start mysql   # macOS
Frontend Build Failed

Error: npm ERR! peer dependency conflict

# Clear cache and reinstall
npm cache clean --force
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
